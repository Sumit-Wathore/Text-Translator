# Text Translator Web App

A simple text translator web application built using React JS and Tailwind CSS.  
It converts English text into multiple Indian languages.

## Features
- Translate English text
- Supports Hindi, Marathi, Bengali, Urdu, and Telugu
- Simple and clean user interface
- Responsive design
- Easy to use

## Tech Stack
- React JS
- Tailwind CSS
- Free Translation API

## How to Run the Project

1. Clone the repository  
   git clone <your-github-repo-link>

2. Go to the project folder  
   cd text-translator

3. Install dependencies  
   npm install

4. Start the project  
   npm run dev

## Purpose of the Project
This project was created as part of my internship to practice React fundamentals, API handling, and UI development using Tailwind CSS.

## Output
Users can enter text in English, select a language, and get the translated output instantly.
