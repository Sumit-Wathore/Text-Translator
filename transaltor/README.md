# Text Translator Web Application

A React-based web application for translating English text to Indian languages.

## Features
- Translate English to Hindi, Marathi, Bengali, Urdu, and Telugu
- Clean and responsive UI with Tailwind CSS
- Two-page navigation (Home and About)
- Real-time translation using MyMemory API
- Character count display
- Error handling
