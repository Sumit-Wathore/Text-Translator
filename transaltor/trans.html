About This Translator

Project Overview

This Text Translator application is built to demonstrate practical web development skills. It allows users to translate English text into several Indian languages instantly using a free public API.

Features

  • Translate English to 5 Indian languages: Hindi, Marathi, Bengali, Urdu, and Telugu
  • Clean and responsive user interface
  • Real-time translation using public API
  • Character count display and input validation
  • Works on both desktop and mobile devices
